# Astra User Flows

## Flow 1 – Panic Alert

1. User opens the Astra app
2. User presses the Panic Alert button
3. The system captures the user's current location
4. The AI system evaluates current risk level
5. A panic alert is created
6. Trusted Safety Circle contacts are notified
7. The user may also choose to call 911 directly

---

## Flow 2 – Safety Check-In

1. User activates Safety Mode
2. The system starts a timer
3. After the timer expires, Astra asks:
   "Are you safe?"
4. The user can respond:
   - I'm Safe
   - Need Help
5. If there is no response:
   - risk score increases
   - alert escalation begins

---

## Flow 3 – Location Sharing

1. User selects "Share Location"
2. User chooses Safety Circle contacts
3. Location updates are shared temporarily
4. Contacts can view the live session
5. User ends sharing when activity is complete

---

## Flow 4 – AI Risk Detection

1. User activates Safety Mode
2. Astra collects location and behavior data
3. The AI system analyzes:
   - missed check-ins
   - inactivity
   - time-based risk
   - movement anomalies
4. Risk score increases when abnormal patterns are detected
5. Astra prompts the user if needed
6. If no response is received, escalation begins

---

## Flow 5 – Emergency Evidence Capture

1. User triggers emergency mode
2. Astra starts evidence capture
3. The system records:
   - audio
   - video
   - photos (if available)
4. Evidence is tagged with:
   - timestamp
   - location
5. Evidence is securely uploaded to cloud storage

---

## Flow 6 – Trusted Contact Incoming Alert

1. Trusted contact receives an Astra emergency notification
2. The trusted contact opens the app
3. They see:
   - user name
   - live location
   - risk level
   - evidence status
   - missed check-in status
4. They can:
   - open live location
   - call the user
   - call 911
   - open live chat

---

## Flow 7 – AI Assistant & Voice Input

1. User opens Astra AI Assistant
2. User can type or speak
3. Astra responds with safety prompts
4. If risk is detected, Astra suggests next actions such as:
   - share location
   - call emergency contact
   - trigger alert
