# Astra AI Risk Detection Model

This model is a rule-based risk scoring system designed for MVP.

It evaluates user safety using behavioral and contextual signals.

## Features Used
- movement speed
- check-in response
- time of day
- safety mode activation

## Output
- LOW_RISK
- MEDIUM_RISK
- HIGH_RISK

## Future Improvements
- machine learning models
- anomaly detection (unsupervised learning)
- personalized user patterns
